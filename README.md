📌 Project Overview

BurmudaTech একটি আধুনিক Full Stack Electronics & Technology E-commerce Platform, যেখানে ব্যবহারকারীরা সর্বশেষ প্রযুক্তিপণ্য সহজেই খুঁজে দেখতে, সার্চ করতে, ফিল্টার করতে, তুলনা করতে এবং নিরাপদভাবে ক্রয় করতে পারবে।

এখানে Admin একটি নিরাপদ Dashboard থেকে Product, Category, Brand, User এবং Order সম্পূর্ণভাবে পরিচালনা করতে পারবে।

প্রজেক্টটি সম্পূর্ণ Production Ready Architecture অনুসরণ করে তৈরি হবে।
